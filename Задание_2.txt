Что такое DNS (Domain Name System) - краткое эссе, своими словами.
DNS -это система хранения доменных имен, используется для росшифровки IP - адреса в символьное имя, для упрощения работы человека в сети, 
т.к. человеку проще запомнить символьное имя, нежели набор цифр.
HTTP - протокол передачи данных.
HTTPS - безопасное расширение протокола HTTP.
HTTP использует порт 80, HTTPS — порт 443.
При передаче данных, использование HTTP протокола может превести к компроментации Ваших личных данных, а также их воровству.
HTTPS спользует шифрование данных, таки образом HTTPS расширение не позволит личным данным пользователя быть украденными.
Версии HTTP/HTTPS:
1. HTTP/0.9 - В марте 1991 был создан механизм для доступа к документам в интернете по средствам использования гипертекста (текстовая страница, имеющая ссылки).
2. HTTP/1.0 - В мае 1996 был выпущен информационный документ, который помог реализовать новую версию HTTP/1.0.
3. HTTP/1.1 - Версия принята в июне 1999. В данной версии появилась возмоность отправлять несколько запросов за одно соеднение.
4. HTTP/2 - 11 февраля 2015 года опубликовала вторая масштабная версия протокола, являющаяся бинарной (двоичная система счисления). 
Особенности версии - сжатие заголовков, поддержка push-уведомлений,передача нескольких потоков данных по одному каналу связи.
